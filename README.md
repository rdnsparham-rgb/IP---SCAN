## 🚀 معرفی پروژه

**IP-SCAN** یک ابزار تحت وب برای دریافت و نمایش اطلاعات مربوط به آدرس‌های IP است که با رابط کاربری ساده و مدرن طراحی شده است.

این پروژه به صورت **Open Source** منتشر شده و توسعه‌دهندگان می‌توانند آن را مشاهده، تغییر و بهبود دهند.

---

## ✨ امکانات

- 🌐 بررسی اطلاعات IP
- 📍 نمایش موقعیت جغرافیایی IP
- 🗺️ نمایش مکان روی نقشه
- 🏢 نمایش ISP و اطلاعات شبکه
- 📋 کپی سریع اطلاعات
- 🔗 اشتراک‌گذاری نتایج
- ⚡ سرعت بالا و رابط کاربری ساده
